# ShopEZ - MERN Stack E-Commerce Application

## Project Overview

ShopEZ is a full-stack e-commerce application developed using the MERN stack.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features

- User Authentication
- Product Catalog
- Product Search & Filtering
- Shopping Cart
- Order Management
- Admin Dashboard

## Project Structure

- client/
- server/

## Installation

Frontend:
cd client
npm install
npm run dev

Backend:
cd server
npm install
npm start
