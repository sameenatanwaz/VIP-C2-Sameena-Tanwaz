##ShopEZ - MERN Stack E-Commerce Application
##Project Overview

ShopEZ is a full-stack e-commerce application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application enables users to browse products, manage carts, place orders, and provides administrative functionality for product and order management.

##Tech Stack
React.js
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication
##Features
User Authentication
Product Catalog
Product Search & Filtering
Shopping Cart
Order Management
Admin Dashboard
Responsive UI
